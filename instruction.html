How to Modify the Travelo Template

1. Overview

Welcome to the Travelo Tours and Travel Website Template! This guide will help you customize the template to suit your travel business needs. Follow each section carefully to make the most of this template.

2. Editing the Homepage

To modify the homepage:

  • Open the home1.html file in your preferred text editor.
  • Locate sections such as the hero , Package listing, hotel listing, gallery ,banner and testimonials. These can be identified by their respective comment names like hero, package listing, hotel listing ,gallery ,banner , review.
  • Update the text, images, and links as needed. Ensure that you maintain the structure of the HTML for a consistent layout.
  • Save your changes and preview them in a browser.
  • Similarly you can change the other home page .

3. Customizing Package Listings

To customize the package listings page:

  • Open the package.html file.
  • Each tour listing is wrapped in a div with a class like .tour-item. Update the tour name, description and image.
  • If you need to add a new tour, duplicate an existing tour-item block and modify the content accordingly.
  • Save your changes and check the updated listings on your site.

4. Customizing Package Description

To customize the package description page:

  • Open the package-description.html file.
  • You Can Change the main description .
  • Each day is wrapped in a div with a class like .day. Update the Package day, description and image.
  • If you need to add a day tour, duplicate an existing day block and modify the content accordingly.
  • Save your changes and check the updated listings on your site.

5. Customizing Hotel Listings

To customize the hotel listings page:

  • Open the hotel.html file.
  • Each hotel listing is wrapped in a div with a class like .hotel. Update the hotel name, description and image.
  • If you need to add a new hotel, duplicate an existing hotel block and modify the content accordingly.
  • Save your changes and check the updated listings on your site.

6. Customizing Hotel Details

To customize the hotel detail page:

  • Open the view-hotel-detail.html file.
  • Each hotel listing is wrapped in a div with a class like .detail. Update the hotel name, description, price and image.
  • If you need to add a new hotel detail, duplicate an existing detail block and modify the content accordingly.
  • Save your changes and check the updated listings on your site.

7. Customizing the Booking Form

To modify the booking form:

  • Navigate to the payment.html file.
  • Locate the form within the form tag class.
  • Update form fields as necessary. For example, if you need additional fields like "Number of Travelers" or "Preferred Travel Date," you can add new input elements.
  • Ensure any new fields are appropriately styled to match the existing design.
  • Test the form to make sure all inputs are functioning correctly.

8. Updating the Contact Information

To update the contact information:

  • Open the contact.html file.
  • Locate the section containing contact details, typically within a .contact-info class.
  • Update form fields as necessary.
  • Make sure to save your changes and verify them on the live site.

9. Customizing Blog Listings

To customize the blog listings page:

  • Open the blog.html file.
  • Each blog listing is wrapped in a section with a class like .blog. Update the blog place, name, description and image.
  • If you need to add a new blog, duplicate an existing blog block and modify the content accordingly.
  • Save your changes and check the updated listings on your site.

10. Customizing Blog Details

To customize the blog detail page:

  • Open the blog-detail.html file.
  • You Can Change the main description, blog title .

11. Customizing Gallery

To customize the gallery page:

  • Open the gallery.html file.
  • You Can Change the image name by changing the image src .

12. Customizing Food

To customize the food page:

  • Open the food.html file.
  • You Can Change the image name by changing the image src .

13. Customizing the CSS

The template uses Index CSS for styling. To customize the look and feel:

  • Open the main CSS file, often named index.css.
  • You can customize colors, fonts, and spacing by editing classes.
  • Here tailwind and bootstrap framework are used,for modification you can refer tailwind https://tailblocks.cc and bootstrap https://getbootstrap.com
  • Save your changes and check the appearance on different devices to ensure responsiveness.

15. Final Review

After all modifications:

  • Review each page to ensure that the content is properly displayed.
  • Test all links, forms, and interactive elements.
  • Ensure that the template looks good on different devices and screen sizes.
